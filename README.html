Odyssey

Odyssey is an R shiny application for the exploration of Molecular Biodiversity in Greece and Norway.

Table of Contents

  • Installation
  • Usage
  • Contributing
  • License

Installation

# install.packages("remotes")
remotes::install_github("BiodataAnalysisGroup/ELIXIR-BFSP-Odyssey")

Usage

# library(Odyssey)
Odyssey::run_odyssey()
