# UI Module: Home tab

A Shiny UI module that defines the "Home" tab of the Odyssey app
interface. It displays introductory content, such as app information and
welcome text.

## Usage

``` r
home_ui(id)
```

## Arguments

- id:

  A character string used to specify the module namespace.

## Value

A Shiny `nav_panel` containing the "Home" section layout.
