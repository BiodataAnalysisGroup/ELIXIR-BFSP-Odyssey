# Odyssey Shiny Application Server

Defines the server logic of the Odyssey Shiny app.

## Usage

``` r
app_server(input, output, session)
```

## Arguments

- input:

  Shiny input object.

- output:

  Shiny output object.

- session:

  Shiny session object.

## Value

The Shiny server function for the Odyssey app.
