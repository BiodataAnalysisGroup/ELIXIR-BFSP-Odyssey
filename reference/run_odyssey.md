# Run Odyssey Shiny Application

Launches the Odyssey Shiny app for exploring molecular biodiversity. The
app provides interactive tabs for Home, Overview (summary statistics and
visualizations), Table (data exploration and download), and Map
(geographical visualization).

## Usage

``` r
run_odyssey(...)
```

## Arguments

- ...:

  parameters

## Value

A Shiny application object launched in the current R session
