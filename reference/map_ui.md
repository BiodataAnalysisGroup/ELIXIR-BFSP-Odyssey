# UI Module: Map Tab

Defines the user interface for the Map tab of the Odyssey Shiny
application. This tab displays an interactive Leaflet map that
visualizes molecular biodiversity data.

## Usage

``` r
map_ui(id)
```

## Arguments

- id:

  Character string used for namespacing the input IDs in the UI module.

## Value

A `tagList` with UI elements for selecting the data source and filters.
