# UI Module: Header

A Shiny UI module that displays the application header, including the
title, GitHub repository link, and an info button.

## Usage

``` r
header_ui(id)
```

## Arguments

- id:

  A character string used to specify the module namespace.

## Value

A `tagList` containing the header UI elements (title, GitHub link, and
info button).

## Details

This module defines the top section of the Odyssey Shiny app, featuring
a styled title and toolbar with quick access to the GitHub repository
and an information modal.
